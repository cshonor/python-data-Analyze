# 广播机制 (Broadcasting)

NumPy 的广播机制允许不同形状的数组进行运算，无需显式复制数据，自动扩展维度以匹配。

## 核心规则

从**右往左**比较 shape，维度满足以下之一即可广播：
1. **维度相等**
2. **其中一方为 1**（可扩展）

```python
# 例如 (3, 4) 与 (4,) 比较
# (3, 4) vs (4) → 4==4 ✓，然后 (3,) 与空 → 可广播
```

## 常见场景

### 1. 标量与数组

```python
import numpy as np

arr = np.array([1, 2, 3])
print(arr + 10)   # [11 12 13]，10 被扩展为 [10, 10, 10]
print(arr * 2)    # [2 4 6]
```

### 2. 一维与二维

```python
arr = np.array([[1, 2, 3], [4, 5, 6]])  # (2, 3)
row = np.array([10, 20, 30])             # (3,)
print(arr + row)
# [[11 22 33]
#  [14 25 36]]
# row 沿行方向复制成 (2, 3)
```

### 3. 列向量与二维

```python
arr = np.array([[1, 2, 3], [4, 5, 6]])  # (2, 3)
col = np.array([[10], [20]])             # (2, 1)
print(arr + col)
# [[11 12 13]
#  [24 25 26]]
# col 沿列方向复制成 (2, 3)
```

### 4. 形状 (3,) 与 (4, 3)

```python
a = np.array([1, 2, 3])       # (3,)
b = np.ones((4, 3))           # (4, 3)
print((a + b).shape)          # (4, 3)
```

## 广播步骤示意

```
a.shape = (3, 4)
b.shape =     (4)

比较: 4==4 ✓, 3 与空 → 在 b 左侧补 1 得 (1, 4)
b 沿 axis=0 复制 3 次 → (3, 4)
最终: (3, 4) + (3, 4) 可运算
```

## 不兼容示例

```python
a = np.array([1, 2, 3])    # (3,)
b = np.array([1, 2])       # (2,)
# a + b  # ValueError: operands could not be broadcast together
```

(3,) 与 (2,) 从右比较：3 ≠ 2，且无一方为 1，无法广播。

## 要点

- 从 shape 的**右端**对齐比较
- 维度相等或一方为 1 才能广播
- 为 1 的维度会被扩展复制，不额外占用大量内存
- 广播失败会抛出 `ValueError`
