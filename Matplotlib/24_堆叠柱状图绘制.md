# 堆叠柱状图绘制：将多组数据堆叠展示占比

## plt.bar(bottom=...)

第二组用 bottom 堆叠在前一组之上。

## 示例

```python
import matplotlib.pyplot as plt

x = ['A', 'B', 'C', 'D']
y1 = [10, 20, 15, 25]
y2 = [15, 18, 22, 20]
y3 = [5, 12, 8, 15]

plt.bar(x, y1, label='类别1')
plt.bar(x, y2, bottom=y1, label='类别2')
plt.bar(x, y3, bottom=[a+b for a,b in zip(y1,y2)], label='类别3')

plt.legend()
plt.show()
```

## 使用 np 简化

```python
import numpy as np
y1, y2, y3 = np.array(y1), np.array(y2), np.array(y3)
plt.bar(x, y1)
plt.bar(x, y2, bottom=y1)
plt.bar(x, y3, bottom=y1+y2)
```

## 要点

`bottom` 为前一组的累加高度，多组时需逐层相加。
