# 图形设置和标题重叠：解决多子图标题、标签重叠问题

## plt.tight_layout()

自动调整子图参数，减少重叠，**最常用**。

```python
fig, axes = plt.subplots(2, 2)
# ... 绘制内容 ...
plt.tight_layout()
plt.show()
```

## fig.subplots_adjust()

手动设置子图边距和间距：

```python
fig.subplots_adjust(left=0.1, right=0.9, top=0.9, bottom=0.1, hspace=0.3, wspace=0.3)
```

| 参数 | 说明 |
|------|------|
| left, right, top, bottom | 子图区域相对 Figure 的边距 |
| hspace | 子图之间的垂直间距 |
| wspace | 子图之间的水平间距 |

## 常见重叠场景与对策

| 问题 | 解决 |
|------|------|
| 子图标题重叠 | 增大 `hspace` |
| 纵轴标签重叠 | 增大 `wspace` 或 `left` |
| 总标题与子图重叠 | 减小 `top` 或使用 `fig.suptitle(..., y=1.02)` |
| 底部 x 轴标签被裁切 | 增大 `bottom` |

## 示例

```python
fig, axes = plt.subplots(3, 1, figsize=(8, 9))
fig.subplots_adjust(hspace=0.4)  # 增加垂直间距
fig.suptitle('总标题', y=1.02)
plt.show()
```
