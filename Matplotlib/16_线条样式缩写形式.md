# 线条样式缩写形式：使用简洁语法快速设置线条样式

## 格式

fmt = [marker][line][color]，第三个参数组合颜色、线型、标记。

## 示例

```python
plt.plot(x, y, 'r')      # 红色实线
plt.plot(x, y, 'g--')    # 绿色虚线
plt.plot(x, y, 'b:')     # 蓝色点线
plt.plot(x, y, 'ko-')    # 黑色实线+圆点
plt.plot(x, y, 'r^--')   # 红色虚线+三角
```

颜色 r g b c m y k w。线型 - -- -. :。标记 . o v ^ s * 等。
