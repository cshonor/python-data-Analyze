# 问题解答：常见问题汇总与解决方案

## 1. 中文显示为方框

**原因**：默认字体不支持中文。

**解决**：
```python
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False
```

## 2. 负号显示异常

**解决**：
```python
plt.rcParams['axes.unicode_minus'] = False
```

## 3. 图表不显示

**原因**：未调用 `plt.show()`，或在不支持 GUI 的环境中运行。

**解决**：
```python
plt.show()  # 必须调用
# Jupyter 中可用 %matplotlib inline
```

## 4. 保存的图片为空白

**原因**：在 `plt.show()` 之后保存，图形已被清空。

**解决**：先 `savefig()` 再 `show()`：
```python
plt.savefig('out.png')
plt.show()
```

## 5. 多子图标题或标签重叠

**解决**：
```python
plt.tight_layout()  # 自动调整子图间距
# 或
fig.subplots_adjust(hspace=0.3, wspace=0.3)
```

## 6. 图例被遮挡

**解决**：
```python
plt.legend(loc='best')
# 或调整图例位置、透明度
```
