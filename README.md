# Python 数据分析学习笔记

NumPy、Pandas、Matplotlib 等数据分析相关示例与笔记。

## 项目结构

- **Numpy/**：ndarray、级联、拆分、运算、广播、聚合、查找排序等
- **Pandas/**：Series、DataFrame、pandas-notes 学习大纲
- **Matplotlib/**：绘图基础、柱状图、散点图、饼图、箱线图等
- **Scipy/** / **statsmodels/**：统计与科学计算

## 安装

```bash
pip install numpy pandas matplotlib
# 或
conda install numpy pandas matplotlib
```

## 运行

进入对应目录运行示例脚本，如：

```bash
python Numpy/01.创建ndarray.py
python Numpy/04.ndarray读写.py
```
