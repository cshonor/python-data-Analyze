# NumPy 示例项目

这是一个NumPy基础示例项目，展示了NumPy库的主要功能。

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 功能示例

项目包含以下NumPy功能示例：

- 数组创建（一维、二维、特殊数组）
- 数组属性（形状、维度、大小、数据类型）
- 数组运算（加减乘除、点积）
- 数组索引和切片
- 数组形状操作（重塑、展平）
- 数学函数（平方根、三角函数、指数、对数）
- 统计函数（均值、中位数、标准差等）
- 随机数生成
- 矩阵运算（矩阵乘法、转置、行列式、逆矩阵）
