# 03 Series 运算与缺失值（对齐 / isna / fillna）
  
量化数据里最常见的现实：**不同来源数据对不齐 + 缺失值多**。Series 的运算语义（按 index 对齐）和缺失值处理，是你写策略能否稳定的关键。
  
## 1. 运算：按索引对齐
  
- `s1 + s2`：会按 index 对齐
- 不匹配的 index 会变成 `NaN`
  
这在多资产收益率、因子合并时非常常见。
  
## 2. 缺失值识别
  
- `s.isna()` / `s.notna()`
- `s.dropna()`：直接丢掉缺失值（要谨慎，可能改变样本）
  
## 3. 缺失值填充（常用）
  
- `fillna(0)`：缺失当 0（要有业务含义）
- `ffill()`：向前填充（时间序列常用）
- `bfill()`：向后填充（少用）
  
## 4. 量化常见用法
  
- 价格序列缺失：通常 `ffill`（停牌/缺失日）
- 因子缺失：可能填 0、行业中性后填均值、或直接剔除（视策略而定）
