# 01 Series 基础（创建 / 索引 / 属性 / 对齐）
  
Series 是 Pandas 的一维数据结构：**数据(values) + 索引(index)**。它既像“带标签的 ndarray”，也像“单列的表”。
  
## 1. 创建 Series（常见三种）
  
- **列表/ndarray 创建**：默认索引是 0,1,2,...
- **字典创建**：字典的 key 自动成为索引
- **标量创建**：配合 index 进行广播填充
  
## 2. 索引与访问
  
- `s['a']`：按标签取值（推荐在标签索引里使用）
- `s.iloc[0]`：按位置取值（0-based）
- `s.loc['a']`：按标签取值（更明确）
  
## 3. 常用属性
  
- `s.values`：底层 ndarray（计算更快，但丢失标签语义）
- `s.index`：索引对象
- `s.dtype / s.shape / s.size / s.name`
  
## 4. 对齐（量化里非常重要）
  
Series 之间运算会 **按索引对齐**，这是 Pandas 的核心语义：
  
- 对齐不到的索引会产生 `NaN`
- 这对多资产、多因子合并非常关键
  
## 5. 新手最容易踩坑
  
- 不要把 `values` 当成万能：一旦转成 ndarray，就失去索引对齐能力
- 做策略时要保留 index（日期）以避免“对不齐却没发现”的隐形错误
