# 05 DataFrame 选择与索引（loc / iloc / 条件过滤）
  
DataFrame 的取数核心就三类：**按标签（loc）**、**按位置（iloc）**、**按条件（布尔过滤）**。量化里你会每天做：按日期切片、按字段取列、按条件筛选信号。
  
## 1. `loc`：按标签
  
- 行标签通常是日期：`df.loc['2025-01-01']`
- 行切片：`df.loc['2025-01-01':'2025-01-31']`（包含终点）
- 行列同时选：`df.loc['2025-01-01':'2025-01-31', ['close','volume']]`
  
## 2. `iloc`：按位置
  
- `df.iloc[0]`：第一行
- `df.iloc[:, 0]`：第一列
- `df.iloc[0:5, [0,3]]`：前 5 行、取第 0/3 列
  
## 3. 条件过滤（信号/选股常用）
  
- `df[df['close'] > df['open']]`：阳线日
- `df.query('close > open')`：同样效果（语法更简洁）
  
## 4. 常见坑
  
- `df[0:5]` 是按“行位置切片”，但不够明确；推荐统一用 `iloc`
- `loc` 的日期切片需要 index 是排序的（时间序列通常是）
