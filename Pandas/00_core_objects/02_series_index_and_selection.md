# 02 Series 索引与选择（loc / iloc / 布尔过滤）
  
这一节把 Series 的“取数方式”一次讲清楚：**按标签**、**按位置**、**按条件**。
  
## 1. `loc`：按标签取
  
- `s.loc['a']`：取标签为 `'a'` 的元素
- `s.loc[['a','c']]`：取多个标签（返回 Series）
- `s.loc['a':'d']`：标签切片（**包含**终点，和 Python 切片不同）
  
## 2. `iloc`：按位置取
  
- `s.iloc[0]`：取第 0 个
- `s.iloc[[0,2]]`：取多个位置
- `s.iloc[0:3]`：位置切片（**不包含**终点，和 Python 一致）
  
## 3. 布尔过滤（量化最常用）
  
- `s[s > 0]`：筛选满足条件的子集
- 条件要能与索引对齐（通常直接来自同一 Series/同 index 的 Series）
  
## 4. 常见坑
  
- `s['a':'d']` 的行为取决于索引类型与排序情况；推荐明确写 `loc/iloc`
- `loc` 的标签切片包含终点，`iloc` 的位置切片不包含终点
