# Pandas 示例

## 1.1 什么是 Pandas

- Pandas 是 Python 的扩展程序库，用于**数据分析**
- 开源、BSD 许可，提供**高性能、易用**的数据结构与数据分析工具
- 名字源于 "**panel data**"（面板数据）和 "**Python data analysis**"
- 基础是 **NumPy**（高性能矩阵运算）
- 可导入 CSV、JSON、SQL、Excel 等多种格式
- 支持归并、重塑、选择、**数据清洗**与加工

Pandas 的核心数据结构是 **Series**（一维）与 **DataFrame**（二维），足以处理金融、统计、社会科学、工程等领域的大多数典型用例。

## 主要功能

- **Series**：一维 labeled 数组
- **DataFrame**：二维表格数据结构
- 数据读取与导出（CSV、Excel、JSON 等）
- 数据清洗与预处理
- 数据筛选、分组、聚合
- 时间序列处理
